# Financial Monitoring Platform

A personalized stock market dashboard that enables users to follow stocks, build a watchlist, and receive smart in-app alerts.

## Planned Stack
- Frontend: React + TypeScript + Chakra UI
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL + Prisma
- External API: Finnhub
- Deployment: Render + Render Postgres

## Project Structure
- `frontend/` - React application
- `backend/` - Express API
- `shared/` - shared types/constants
- `docs/` - project specification and documentation
